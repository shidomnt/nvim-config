# Vim Config 

[Note]
- Can run on both Window and Linux
- Run these commands after pulling this resource
:PackerCompile
:PackerClean
:PackerInstall
:PackerUpdate

[Requirement]
- NodeJS
- Python
- Git
- vim-plug
- ripgrep
- Gcc lastest (mingw)

[Language Servers]
- Install by :LspInstall <language_to_install>

[Language parsers]
- Install by :TSInstall <language_to_install>
