# Vim Config 

[Note]
- Can run on both Window and Linux
- Using packer to manage plugins

[Requirement]
- NodeJS
- Python
- Git
- vim-plug
- ripgrep
- Gcc lastest (mingw)

[Language Servers]
- Install by :LspInstall <language_to_install>

[Language parsers]
- Install by :TSInstall <language_to_install>
